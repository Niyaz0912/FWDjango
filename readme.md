# Название проекта

Web320Shelter

## Статус

Проект находится в стадии разработки.

## Установка

1. Клонируйте репозиторий.
2. Установите зависимости с помощью `pip install -r requirements.txt`.
3. Запустите приложение командой `python manage.py runserver`.

## Требования

- Python 3.x
- Django 3.x

## Технологии

- Django
- MS SQL Server
